# 明DAO - 真实梵高《星空》K线背景使用指南

## ✨ 功能说明

"关于我们"页面已经集成了**真实梵高《星空》的K线版本**！每个像素点都由一根动态波动的K线替代，创造出独特的艺术效果。

## 🎯 如何使用真实的《星空》图片

由于跨域限制，您需要手动下载图片到项目中：

### 步骤1：下载《星空》原图

访问以下网址并下载图片：
```
https://uploads2.wikiart.org/images/vincent-van-gogh/the-starry-night-1889(1).jpg
```

或者直接在浏览器中右键点击并"图片另存为"。

### 步骤2：重命名并放置图片

1. 将下载的图片重命名为：`starry-night.jpg`
2. 将图片移动到项目的 `public` 文件夹：
   ```
   C:\Users\marcu\Desktop\网站建设\DAO1\public\starry-night.jpg
   ```

### 步骤3：重启开发服务器

在命令行中停止服务器（Ctrl+C），然后重新运行：
```bash
npm run dev
```

### 步骤4：查看效果

访问 `http://localhost:3000/about`，您将看到：
- **真实星空**按钮 - 使用真实的梵高《星空》图片（如果您已按上述步骤添加图片）
- **轮廓描边**按钮 - 只绘制轮廓的版本
- **密度映射**按钮 - 用K线密度表现明暗的版本

## 🔄 备用方案

如果您没有添加图片，系统会自动使用**程序化生成的星空数据**作为备用方案，效果也很不错！

## 🎨 三个版本的区别

| 版本 | 说明 | K线数量 | 背景色 |
|------|------|---------|--------|
| **真实星空** | 100x70像素网格，每个像素一根K线，完全还原原画色彩 | ~7000根 | 黑色 |
| **轮廓描边** | 提取轮廓线条，用K线勾勒形状 | ~2000根 | 深蓝渐变 |
| **密度映射** | 亮处K线密集，暗处稀疏 | ~5000根 | 深蓝渐变 |

## 🖱️ 交互效果

- **点击画面任意位置**：产生涟漪效果，K线会从点击点向外扩散波动
- **动态波动**：每根K线每秒随机波动一次，模拟真实市场

## 🎯 色彩映射说明

原画的色彩被映射到交易平台的配色：
- **亮黄色**（星星、月亮） → 金黄色 `#f4d03f`
- **蓝色**（天空） → 深蓝 `#4a6fa5` / 浅蓝 `#5d9cec`
- **深色**（丝柏树、山脉） → 深灰/黑 `#1a1a1a`
- **其他亮色** → 朱红 `#C83C23`

## 📱 移动端

所有三个版本都完美适配移动端，K线数量会根据屏幕尺寸自动调整。

## ⚡ 性能优化

- K线分批更新：每秒更新所有K线，但分60批进行，保证流畅度
- 响应式K线宽度：自动适应屏幕大小

## 🐛 故障排查

**问题：页面显示"使用程序化生成的星空数据"**
- 说明图片未成功加载
- 请检查图片是否放在正确的位置：`public/starry-night.jpg`
- 确保图片文件名正确（全小写，包含连字符）

**问题：页面加载很慢**
- 正常现象，首次提取7000个像素点需要一些时间
- 加载后会显示平滑的动画
